# Grid-gustos - Iván Sánchez

Este proyecto consiste en un portfolio personal estilo "Moodboard" diseñado con una estructura de rejilla dinámica y un sistema de cambio de tema (claro/oscuro) sin dependencia de JavaScript.

## 👤 Autor
* **Nombre:** Iván Sánchez
* **Curso:** 1º DAM (Desarrollo de Aplicaciones Multiplataforma)

## 📝 Descripción de la Práctica
La práctica se centra en la creación de un diseño web adaptativo (Responsive Design) utilizando **CSS Grid** para la organización de elementos y **CSS Flexbox** para la alineación de componentes internos. 

### Características principales:
* **Grid Layout Avanzado:** Uso de `grid-template-areas` para una organización visual limpia y estructurada.
* **Dark Mode Nativo:** Implementación del cambio de tema mediante el selector de CSS `:has()` y variables raíz (`:root`), permitiendo alternar entre modo claro y oscuro solo con CSS.
* **Animaciones:** Sistema de entrada en cascada mediante `@keyframes` y `animation-delay` para mejorar la experiencia de usuario (UX).
* **Responsive Design:** Adaptación completa a dispositivos móviles y tablets mediante Media Queries.

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructura semántica del documento.
* **CSS3:** * Variables CSS (Custom Properties).
    * CSS Grid y Flexbox.
    * Animaciones y transiciones suaves.
    * Selector avanzado `:has()`.
